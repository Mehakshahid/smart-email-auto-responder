# Smart Email Auto-Responder System

## 📌 Project Overview
This project is an AI-powered email automation system that reads incoming emails, classifies them, generates responses, and stores interactions automatically.

## 🚀 Features
- Email trigger using Gmail
- AI-based classification (Support, Sales, Spam)
- Automated email reply generation
- Spam filtering
- Data storage in Google Sheets

## 🛠 Tech Stack
- Make (Automation)
- OpenRouter API (AI)
- Gmail
- Google Sheets

## 🔄 Workflow
Gmail → Make → AI → JSON Parser → Filter → Auto Reply → Google Sheets

## 🧠 How It Works
1. Email is received in Gmail
2. Make triggers workflow
3. Email content is sent to AI (OpenRouter)
4. AI classifies and generates reply
5. JSON parser extracts data
6. System sends reply automatically
7. Data is stored in Google Sheets
